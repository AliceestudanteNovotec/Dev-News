1. Instalar o XAMPP

  1. Baixar o XAMPP:
    • Acesse o site oficial do XAMPP: Apache Friends.
    • Clique no botão de download para a versão apropriada do seu sistema operacional (Windows, macOS, ou Linux).
  2. Instalar o XAMPP:
    • Execute o arquivo de instalação que você baixou.
    • Siga as instruções do assistente de instalação. Você pode optar por instalar todos os componentes ou apenas os necessários (Apache e MySQL são os essenciais).
  3. Iniciar o XAMPP:
    • Abra o XAMPP Control Panel.
    • Clique em "Start" ao lado dos serviços "Apache" e "MySQL" para iniciar o servidor web e o banco de dados local.

2. Configurar o XAMPP

  1. Verificar se o Apache e MySQL estão funcionando:
    • Abra seu navegador e digite http://localhost na barra de endereços. Você deve ver a página inicial do XAMPP, confirmando que o Apache está funcionando.
  2. Acessar o phpMyAdmin:
    • No navegador, digite http://localhost/phpmyadmin e pressione Enter. Isso abrirá o phpMyAdmin, uma interface gráfica para gerenciar bancos de dados MySQL.

3. Criar um Banco de Dados

  1. Criar um novo banco de dados:
    • No phpMyAdmin, clique na aba "Databases" (ou "Bases de Dados").
    • No campo "Database name" (ou "Nome do banco de dados"), digite o nome do banco de dados, por exemplo, vendas_noivas.
    • Clique no botão "Create" (ou "Criar").

4. Criar uma Tabela

  1. Selecionar o banco de dados:
    • Clique no nome do banco de dados vendas_noivas na lista à esquerda.
  2. Criar uma nova tabela:
    • Na seção "Create table" (ou "Criar tabela"), digite o nome da tabela, por exemplo, vestidos.
    • Defina o número de colunas, por exemplo, 3, e clique em "Go" (ou "Ir").
  3. Definir a estrutura da tabela:
    • Preencha os campos da tabela como segue:
      • Coluna 1:
        • Name: id
        • Type: INT
        • Length/Values: (deixe em branco ou defina como 11)
        • Index: PRIMARY
        • A_I (Auto Incremento): Marque essa caixa para que o id seja gerado automaticamente.
      • Coluna 2:
        • Name: cor
        • Type: VARCHAR
        • Length/Values: 50 (ou o comprimento desejado)
      • Coluna 3:
        • Name: tamanho
        • Type: VARCHAR
        • Length/Values: 10 (ou o comprimento desejado)
    • Após preencher todos os campos, clique em "Save" (ou "Salvar").

5. Confirmar a Criação da Tabela

  • Sua tabela vestidos deve agora aparecer na lista de tabelas dentro do banco de dados vendas_noivas. Você pode começar a adicionar e gerenciar dados clicando em "Browse" (ou "Navegar") para visualizar ou em "Insert" (ou "Inserir") para adicionar novos registros.
